# 微信小程序简历

用微信开发者工具做的个人简历微信小程序，纯 wxml + wxss 完成
所有个人文字介绍都在 resume.config.js 配置

## 效果演示

微信搜索 『**王寻兵的个人简历**』，或者直接扫码

![](./img/qrcode.jpg)
